# Dev Burger

## Visão Geral

O projeto apresenta:
- Hero section com chamada principal
- Seção de cardápio com produtos e preços em AOA
- Bloco "Sobre" com informações da marca
- Chamada para pedido via WhatsApp
- Rodapé com contato, horário e redes sociais

## Funcionalidades

- Navegação por âncoras (Início, Cardápio, Sobre e Contato)
- Cardápio com destaques e preços em AOA
- Botões de pedido com redirecionamento para WhatsApp
- Seção institucional sobre a marca
- Layout responsivo para diferentes tamanhos de tela

## Stack

- HTML5
- CSS3

## Estrutura do Projeto

```text
.
|-- index.html
|-- styles.css
|-- README.md
`-- img/
```

## Como Executar

1. Clone este repositório:

```bash
git clone <URL_DO_REPOSITORIO>
```

2. Entre na pasta do projeto:

```bash
cd Dev\ Burger
```

3. Abra o arquivo `index.html` no navegador.

## Personalização

- WhatsApp: substitua `https://wa.me/244929999999` pelo número real.
- Cardápio: edite títulos, descrições e preços na seção `#cardapio` em `index.html`.
- Marca e contatos: ajuste conteúdos em `#sobre`, `#cta` e `footer`.
- Imagens: troque os arquivos na pasta `img/` ou atualize os caminhos no HTML.
